一、设置开发环境

1、node官网下载安装node,安装后查看node版本命令： node -v

2、然后全局安装Angular CLI,命令 npm install -g @angular/cli,安装后查看CLI版本命令: ng -v

若已经安装过Angular CLI,可自行选择更新Angular CLI,
更新方法:卸载Angular CLI命令: npm uninstall -g angular-cli
清除缓存: npm cache verify
安装新版本(全局): npm install -g @angular/cli

二、创建新项目

项目目录下命令: ng new my-app
